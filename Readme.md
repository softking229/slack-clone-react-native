## How to run project

```sh
git clone git@github.com:GetStream/slack-clone-react-native.git
cd slack-clone-react-native

# Install npm dependencies
yarn

# Install pod dependencies
cd ios && pod install && cd ..

# Run on ios simulator
react-native run-ios
```
